# prchecklist

> A GitHub App built with [Probot](https://github.com/probot/probot) that Puts a comment on PRs with a checklist. Allows merge once all the mandatory checklists are checked.

## Setup

```sh
# Install dependencies
npm install

# Run the bot
npm start
```
